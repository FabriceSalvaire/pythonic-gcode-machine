4. Installation¶

Note to Packagers: Please don’t create PythonicGcodeMachine package (PiPY do the job)

4.1. On Windows¶

Firstly, you have to install the Anaconda Distribution so as to get a full featured Python 3 environment.

Then open the Anaconda Navigator and launch a console for your root environment.

You can now run pip to install PythonicGcodeMachine in your root environment using this command:

pip install PythonicGcodeMachine

4.2. On Linux¶

Firstly, you have to install Python 3 from your distribution.

Then you can install PythonicGcodeMachine using pip or from source. See supra.

4.3. On OSX¶

There are several ways to get Python on OSX:

  • use the built in Python
  • install Miniconda
  • install the Anaconda Distribution.
  • install from Brew brew install python3 (reported to work)

You can install PythonicGcodeMachine using pip or from source. See supra.

4.4. Installation from PyPi Repository¶

PythonicGcodeMachine is available on the Python Packages PyPI repository at https://pypi.python.org/pypi/PythonicGcodeMachine

Run this command in the console to install the latest release:

pip install PythonicGcodeMachine

4.5. Install a more recent version from Github¶

If you want to install a version which is not yet released on Pypi, you can use one of theses commands to install the stable or devel branch:

pip install git+https://github.com/FabriceSalvaire/PythonicGcodeMachine

pip install git+https://github.com/FabriceSalvaire/PythonicGcodeMachine@devel

4.6. Installation from Source¶

The PythonicGcodeMachine source code is hosted at https://github.com/FabriceSalvaire/PythonicGcodeMachine

You have to solution to get the source code, the first one is to clone the repository, but if you are not familiar with Git then you can simply download an archive either from the PythonicGcodeMachine Pypi page (at the bottom) or the GitHub page (see clone or download button).

To clone the Git repository, run this command in a console:

git clone https://github.com/FabriceSalvaire/PythonicGcodeMachine.git

Then to build and install PythonicGcodeMachine run these commands:

python setup.py build
python setup.py install
